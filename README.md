# lantern-template


> lantern 模板

## Usage

This is a project template for [lantern-cli](https://github.com/ssfe-team/lantern-cli).

``` bash
$ npm install -g lantern-cli
$ lantern init lantern-template my-project
$ cd my-project
$ npm install
$ npm run dev
```
